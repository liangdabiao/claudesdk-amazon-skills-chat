# 亚马逊商品图片策划

## 概述

本技能为亚马逊卖家提供系统化的商品图片策划服务，帮助卖家规划和设计能够最大化转化率的商品图片组合。在亚马逊平台上，商品图片是消费者了解产品的第一窗口，图片质量直接决定了点击率和转化率的高低。尤其对于无法触碰实物、仅靠屏幕信息做购买决策的电商场景而言，图片承担了"视觉销售员"的核心角色。

本技能的核心价值在于将商品图片策划从"凭感觉排图"升级为"基于数据和策略的系统化规划"。通过明确的拍摄需求清单（Shot List）、信息图表布局设计、生活场景规划和移动端优化方案，确保每一张图片都在消费者转化链路中发挥明确的战略作用。

## 核心原理

### 七张图片战略框架

亚马逊为每个商品提供最多九张图片位，但本技能采用"七张核心图片"的战略框架进行规划，因为前七张图片（主图加六张副图）是消费者在首屏和首次滚动中最可能看到的内容，对转化的影响最为直接：

- **第一张：主图（Hero Image）**——主图承担着两重任务：在搜索结果页面吸引点击，在商品详情页建立第一印象。主图必须严格遵守亚马逊的图片规范（纯白背景、产品占画面85%以上、无文字无水印），同时通过灯光、角度和构图让产品呈现最具吸引力的视觉效果。本技能会根据产品类型提供具体的主图拍摄指导，例如反光产品需要柔光箱消除高光、透明产品需要特定背景处理来展示通透感、小型产品需要微距镜头呈现细节质感等。
- **第二张：多角度展示**——提供消费者在主图中无法看到的视角（背面、侧面、底部、内部等），消除消费者对产品外观的未知感和疑虑。本技能根据产品的物理特征选择最有信息价值的角度组合。
- **第三张：核心卖点信息图**——将产品的两到三个最核心卖点以图文结合的方式可视化呈现。本技能遵循"一图一卖点"的简化原则，避免信息过载导致消费者跳过。关键设计要素包括简洁的图标、有力的短句和清晰的视觉指引线。
- **第四张：功能细节信息图**——深入展示产品的功能性细节，如材质特写、接口展示、结构拆解等。本技能通过标注线和放大镜效果将消费者的注意力引导至关键细节区域。
- **第五张：尺寸规格参考图**——解决消费者"这个产品到底有多大"的疑问。本技能通过将产品与消费者熟悉的事物（如手掌、硬币、手机等）并排展示，或者提供精确的尺寸标注图，帮助消费者建立对产品实际大小的直觉感知。
- **第六张：使用场景生活图**——将产品置于真实的使用场景中，帮助消费者想象拥有和使用该产品的生活画面。本技能根据目标客群的特征（年龄、性别、生活方式、使用环境）精心设计场景元素和模特选择方案。
- **第七张：包装与附件展示图**——展示产品的包装设计和包含的所有附件，消除消费者对"收到货后里面有什么"的疑虑，同时通过精美的包装展示传递品牌质感。

### 信息图表设计原理

信息图表（Infographic）是副图中转化效果最强的类型之一，其设计需要遵循视觉传达的科学原理：

- **信息层级设计**：消费者的视觉注意力是有限的，一张信息图只能有效传达一到三个核心信息点。本技能通过建立清晰的信息层级——主标题（最大字体，核心卖点）→ 辅助说明（中等字体，补充细节）→ 数据支撑（最小字体，量化参数）——确保消费者在三秒内抓住核心信息。
- **色彩心理学应用**：不同颜色传达不同的情感和含义。本技能根据产品类型和目标客群选择匹配的色彩方案，例如环保类产品倾向于使用绿色系传达自然概念，高端产品倾向于使用深色系和金属色传达品质感。
- **视觉引导路径**：通过箭头、标注线、放大框等视觉元素引导消费者的视线按照预设的路径移动，确保最重要的信息被最先看到，次要信息按重要性递减的顺序被接收。
- **数据可视化策略**：将抽象的产品参数转化为直观的视觉表达。例如，将"续航24小时"转化为一个环形进度图，将"防水等级IPX7"转化为水下使用场景图，将"轻至180克"转化为与常见物品的重量对比图。

### 生活场景规划逻辑

生活场景图片的核心目标是触发消费者的情感共鸣和身份认同：

- **目标客群镜像**：场景中的模特和人物应该与目标客群高度匹配。如果产品面向年轻都市女性，场景中出现的应该是穿着时尚的年轻女性在都市环境中的使用画面，而非中年男性在工厂环境中的使用画面。消费者需要在这些画面中"看到自己"。
- **使用场景真实性**：真实的场景比棚拍更具说服力。本技能建议在实际使用环境中拍摄（如厨房中拍摄厨房用具、健身房中拍摄运动装备），而非使用虚假的背景板。
- **痛点-解决方案叙事**：在场景图中隐含"使用前"和"使用后"的对比叙事。例如，一张展示杂乱桌面被产品整理后变得井井有条的图片，比单纯展示产品外观更有说服力。

### 移动端显示优化

鉴于亚马逊平台超过70%的流量来自移动端，商品图片的设计必须以手机屏幕的显示效果为基准：

- **字体可读性保障**：在手机屏幕上（通常宽度不超过430像素），信息图中的文字必须保持足够的字号和对比度。本技能建议移动端信息图中的主标题字号不低于图片短边的5%，确保即使在小屏幕上也清晰可读。
- **竖版构图优先**：手机屏幕是竖向的，竖版构图能够更好地利用屏幕空间。本技能在信息图设计中优先采用竖版比例，避免关键信息被手机屏幕裁切。
- **简化信息密度**：在手机屏幕上，过于复杂的信息图会因为显示面积有限而失去传达效果。本技能建议移动端信息图每张只传达一个核心信息点。
- **缩略图可辨识度**：商品图片在搜索结果中以缩略图形式呈现，本技能确保每张副图在缩小为缩略图时仍然能够让消费者辨认出图片的主要内容，避免使用细节过于丰富的图片作为副图。

## 工作流程

**第一阶段：产品信息与目标客群分析**

收集产品的核心卖点、功能特性、目标客群画像和品牌调性，通过结构化问答确认图片策划所需的所有关键输入。

**第二阶段：七张图片战略规划**

基于产品特征和目标客群，规划七张核心图片的主题、内容重点和视觉风格，生成完整的拍摄需求清单。

**第三阶段：信息图与场景图详细设计**

为每张信息图撰写文案内容、设计布局结构、指定色彩方案和字体风格，为每张场景图描述场景环境、模特特征、道具配置和拍摄角度。

**第四阶段：A/B测试方案与执行指导**

提供图片上线后的A/B测试方案（如测试不同主图的点击率差异、不同卖点排序的转化率差异），并输出设计师可直接执行的设计需求文档。

## 数据来源

无外部数据源。本技能的图片规划基于AI内置知识，包括Amazon商品图片要求规范的公开文档（如白底主图要求、85%画面填充率、最低分辨率标准等）以及电商视觉设计的最佳实践。不依赖任何外部API或第三方数据服务。用户在使用本技能时需要提供产品信息（核心卖点、功能特性、尺寸材质等）和目标受众画像，以便生成符合产品特性的图片策划方案。

## 局限性

- 图片策划方案的质量最终取决于摄影和设计的执行水平。再优秀的策划方案，如果拍摄质量不佳或设计执行不到位，都无法达到预期的转化效果。
- 本技能提供的是设计需求和布局指导，而非直接生成图片素材。卖家需要将这些需求文档交给专业的摄影师和设计师来执行。
- 移动端优化建议基于主流手机屏幕尺寸的平均数据，不同设备和浏览器的实际显示效果可能存在细微差异。
- A/B测试需要足够的流量基数才能得出统计显著性的结论，对于日均访问量较低的产品，测试周期可能较长或结果可信度不足。
- 亚马逊图片规范可能随时间更新（如近年来对主图的要求日趋严格），本技能的规范建议需要根据亚马逊最新的图片政策进行核对。
