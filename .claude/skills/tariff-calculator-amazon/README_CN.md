# Amazon 关税与到岸成本计算器 — 原理详解

## 概述

本工具为跨境电商卖家计算从产地到目的地的**完整到岸成本**（Landed Cost），包括基础关税、Section 301 附加关税、VAT/GST、运费、保险费以及各类杂费。支持多条贸易路线和自由贸易协定优惠。

## 核心计算原理

### 1. 到岸成本公式

到岸成本是商品从工厂到达 Amazon 仓库的总成本：

```
到岸成本 = FOB货值 + 国际运费 + 保险费 + 进口关税 + VAT/GST + 清关费 + 港口费 + 内陆运输费
```

**逐步计算链路**（`calculator.py:246-335`）：

```
FOB货值 (Free on Board)
  + 国际运费 (Freight)
  + 保险费 (Insurance = FOB × 0.3%)
  ────────────────────
= CIF价值 (Cost, Insurance & Freight)

CIF价值
  × 基础关税率
  + CIF × Section 301税率（仅中国→美国）
  ────────────────────
= 关税总额

CIF + 关税
  × VAT/GST税率
  ────────────────────
= 增值税

关税 + VAT + 清关费($150) + 港口费($50)
  ────────────────────
= 税费总额

CIF + 税费总额 + 内陆运输费
  ────────────────────
= 到岸总成本
```

### 2. HS 编码体系

HS（Harmonized System）编码是国际贸易通用的商品分类编码，前 2 位代表**章（Chapter）**，前 4 位代表**税号（Heading）**。

```
HS编码: 8518 30 0000
        ├─── ── ────
        │    │    └── 子目（6-10位，各国自定义）
        │    └────── 税号（国际统一，决定关税率）
        └─────────── 章（国际统一，代表产品大类）
```

`hs_lookup.py` 内置了 24 个常见跨境电商产品的 HS 编码数据库，支持通过**产品描述关键词**和**HS编码**两种方式查询。

**查询优先级**（`hs_lookup.py:311-356`）：
1. 直接输入 HS 编码 → 精确匹配（置信度 100%）
2. 关键词匹配 → 内置映射表（置信度 60-80%）
3. 描述模糊匹配 → 遍历数据库的英/中文描述（置信度 70%）

### 3. 基础关税率

基础关税率按 HS 章节和目的国确定。数据来源为各国公布的**最惠国待遇（MFN）**税率：

| HS章 | 产品类别 | 美国税率 | 欧盟税率 |
|------|----------|----------|----------|
| 39 | 塑料制品 | 6.5% | 6.5% |
| 84 | 机械设备 | 0% | 2% |
| 85 | 电子电气 | 0% | 2% |
| 94 | 家具 | 0% | 0% |
| 95 | 玩具 | 0% | 4.5% |

完整税率表见 `calculator.py:46-58`。

### 4. Section 301 关税（中国→美国）

美国对从中国进口的部分商品征收**额外附加关税**，按 HS 4位税号判定：

| HS税号 | 产品 | 附加税率 |
|--------|------|----------|
| 8471 | 电脑 | 25% |
| 8504 | 电源/变压器 | 25% |
| 8517 | 通讯设备 | 25% |
| 9403 | 家具 | 25% |
| 9503 | 玩具 | 25% |
| 6110/6203/6204 | 服装 | 7.5% |
| 6402 | 鞋类 | 7.5% |

**注意**：Section 301 关税与基础关税**叠加计算**，不是替代关系。例如家具从中国出口到美国的总关税率可能是 0%（基础）+ 25%（301）= 25%。

### 5. VAT/GST 计算

```
VAT基数 = CIF价值 + 关税总额
VAT     = VAT基数 × VAT税率
```

**关键点**：VAT 的计税基数包含关税，形成**税上加税**效应。

| 目的国 | VAT/GST税率 |
|--------|-------------|
| 美国 | 无联邦VAT（各州销售税不同） |
| 德国 | 19% |
| 法国 | 20% |
| 意大利 | 22% |
| 英国 | 20% |
| 加拿大 | 5% GST |
| 澳大利亚 | 10% GST |

### 6. 自由贸易协定优惠

工具支持以下贸易协定的税率减免：

| 协定 | 国家 | 减免幅度 |
|------|------|----------|
| USMCA | 墨西哥/加拿大→美国 | 100% 免税 |
| AUSFTA | 澳大利亚→美国 | 100% 免税 |
| KORUS | 韩国→美国 | 100% 免税 |
| EVFTA | 越南→欧盟 | 50% 减免 |

当检测到符合条件的贸易路线时，自动应用减免（`calculator.py:222-225`）。

### 7. 运费估算

当未提供实际运费时，按重量和运输方式估算：

| 运输方式 | 目的地 | 估算费率 |
|----------|--------|----------|
| 海运 | 美国 | $0.8/kg |
| 空运 | 美国 | $5.0/kg |
| 快递 | 美国 | $8.0/kg |

## 使用方法

```bash
# 基本关税计算
python3 scripts/calculator.py

# 自定义参数
python3 scripts/calculator.py '{
  "hs_code": "8518300000",
  "origin_country": "CN",
  "destination_country": "US",
  "fob_value": 5000.00,
  "quantity": 500,
  "freight_cost": 200.00
}'

# HS编码查询
python3 scripts/hs_lookup.py "wireless earbuds"
python3 scripts/hs_lookup.py "8518" --zh
```

## 数据来源

本技能的所有关税和费率数据均内置在Python脚本中，无任何外部API调用。`hs_lookup.py` 包含一个24个常见跨境电商产品的HS编码数据库，每条记录包含HS编码（6-10位）、中英文产品描述和Section 301附加税率，支持通过产品描述关键词或HS编码两种方式查询。`calculator.py` 中硬编码了三组税率数据：(1) 11个HS章节（Chapter）的基础关税率，涵盖塑料制品（6.5%）、机械设备（0-2%）、电子电气（0-2%）、家具（0-4.5%）、玩具（0-4.5%）等；(2) 14个HS四位税号（Heading）的Section 301附加税率（7.5%-25%），主要针对中国出口到美国的电脑、通讯设备、家具、玩具、服装、鞋类等产品；(3) 6个目的国的VAT/GST税率（德国19%、法国20%、意大利22%、英国20%、加拿大5%GST、澳大利亚10%GST）以及6个自由贸易协定的减免规则（USMCA、AUSFTA、KORUS、EVFTA等）。

用户需提供HS编码、原产国、目的国、FOB货值、数量、运费等输入参数。当未提供运费时，脚本会按运输方式（海运$0.8/kg、空运$5.0/kg、快递$8.0/kg）和目的地进行估算。汇率数据为固定值，不随市场实时变动。HS编码数据库仅覆盖24个常见品类，非常规产品需用户手动输入准确的HS编码和税率。

## 局限性

- 基础关税率为章节级近似值，精确税率需以 HS 10位编码查询
- HS 编码数据库仅覆盖常见品类（24个），非常规产品需手动输入
- 运费估值为简化模型，实际费用受货代报价、航线、季节等因素影响
- 汇率数据为固定值，实际交易按实时汇率结算
- 不包含反倾销税、反补贴税等特殊关税
