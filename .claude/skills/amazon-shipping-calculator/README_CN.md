# Amazon 配送费用计算器

## 概述

配送费用计算器技能帮助卖家准确估算亚马逊FBA（亚马逊物流）和FBM（卖家自发货）模式下的各类配送和仓储费用，包括配送费、月度仓储费、长期仓储费、移除费用等。通过精确的费用计算，卖家可以在FBA和FBM之间做出最优选择，并为产品定价和利润规划提供可靠的成本数据基础。

配送费用是亚马逊产品成本结构中占比较大的组成部分之一，但许多卖家对此缺乏准确的理解。尤其对于尺寸和重量处于分级临界点的商品，微小的尺寸变化可能导致费用级别的跳升，使得看似相似的产品在配送成本上差异显著。该技能的核心价值在于帮助卖家实现费用计算的精确化和配送模式选择的科学化。

## 核心原理

**尺寸重量分级体系**：亚马逊FBA费用体系基于商品尺寸和重量进行分级计算。核心概念包括：

- **实际重量与体积重量**：体积重量 = 长 x 宽 x 高 / 139（以英寸为单位时）或 长 x 宽 x 高 / 5000（以厘米为单位时）。亚马逊取实际重量和体积重量中的较大值作为计费重量。这意味着一个轻但体积大的商品（如抱枕）可能按更高的体积重量计费。
- **尺寸分级**：亚马逊将商品分为标准小件（最长边不超过15英寸，重量不超过12盎司）、标准大件（最长边不超过18英寸，重量不超过20磅）、大件1（最长边不超过60英寸）、大件2（最长边不超过120英寸）和特殊大件。每个级别对应不同的配送费率。
- **分级临界点效应**：当商品的尺寸或重量恰好在分级临界点附近时，微小的变化可能导致费用的大幅跳升。例如，一件18.1英寸长的商品会被归入大件1级别，配送费可能比18英寸长的标准大件高出数美元。该技能会特别标注这种临界点风险，并提供优化建议。

**FBA费用全貌**：完整的FBA费用体系不仅包含配送费，还包括：

- **月度仓储费**：按每月每立方英尺收取，标准尺寸商品通常为每立方英尺0.87美元（1-9月）和2.40美元（10-12月旺季）。大件商品的费率更高。库存存放时间越长，仓储费累积越多。
- **长期仓储费**：库存存放超过271天后，每件收取6.90美元的长期仓储费或每立方英尺0.15美元（取较高者）。存放超过365天后费率翻倍。长期仓储费是滞销库存的"隐形杀手"。
- **移除/处置费**：将库存从亚马逊仓库退回每件收取0.99美元起（标准件），销毁每件收取0.97美元起。需要计算移除费用与长期仓储费之间的经济比较，决定是继续存储还是移除。
- **入库运输费**：使用亚马逊合作承运人（Partnered Carrier）或自行安排运输将商品送入亚马逊仓库的费用。

**FBA与FBM的成本对比模型**：选择FBA还是FBM并非简单的费用对比，而是一个多维度的综合决策：

- **FBA优势**：更高的购物车获取率、更快的配送速度（Prime标识）、更低的订单处理人工成本、亚马逊负责退换货处理。
- **FBM优势**：无月度仓储费和长期仓储费风险、更高的单位利润（无需支付FBA配送费）、对库存周转的完全控制、适合大件重货或周转缓慢的商品。
- **决策关键**：对于周转快（月销量高）的商品，FBA的综合成本效益通常更优，因为高周转避免了长期仓储费，且FBA带来的购物车优势能显著提升销量。对于周转慢的商品或大件重货，FBM可能是更经济的选择。

## 工作流程

1. **产品规格数据收集**：获取商品的精确尺寸（含包装后）、重量、销售价格、预期月销量等基础信息。
2. **尺寸分级确认**：根据商品的尺寸和重量确定亚马逊的尺寸分级和对应的计费重量，识别是否存在分级临界点风险。
3. **FBA全费用计算**：逐项计算配送费、月度仓储费（区分标准月和旺季）、长期仓储费预估、移除/处置费参考以及入库运输费用。
4. **FBM费用估算**：基于商品尺寸重量和目标配送区域，估算主流承运商（USPS、UPS、FedEx等）的配送费用。
5. **对比分析与建议输出**：将FBA和FBM的完整费用结构进行对比，结合购物车影响、运营便利性和库存风险等因素，给出最优配送模式建议。同时提供尺寸优化建议（如修改包装以降低尺寸分级）和仓储策略建议。

## 数据来源

本技能的FBA费用计算参数基于Amazon官方公布的尺寸分级标准和费率表，参考了amazon-fba-calculator的分级体系，涵盖标准小件、标准大件、大件1、大件2及特殊大件等各级别的配送费率。体积重量的计算公式（DIM weight = 长 x 宽 x 高 / 139，以英寸为单位时）采用Amazon官方标准。月度仓储费、长期仓储费、移除处置费等参数均来自Amazon FBA费用帮助页面的公开信息。

费用计算需要用户提供产品物理规格数据，包括产品包装后的精确尺寸（长、宽、高）和重量、预期月销量预估、目标配送区域等。这些输入数据的准确性至关重要——尤其是包装后的尺寸，因为亚马逊按外包装尺寸进行分级计费，如果用户提供的尺寸未包含外包装，计算结果将产生偏差。

FBM运费数据为行业估算值，参考了USPS、UPS、FedEx等主流承运商的公开费率表进行估算，实际费用可能因燃油附加费、偏远地区附加费、淡旺季浮动等因素而有所不同。建议卖家结合实际承运商报价进行最终确认。亚马逊FBA费率会定期调整，本技能使用的费率基于最近一次公布的费率标准，实际执行时请以Seller Central后台显示的最新费率为准。

## 局限性

- 费用计算依赖于用户提供的商品尺寸和重量数据。如果提供的尺寸未包含外包装（如瓦楞纸箱、气泡袋），实际计费尺寸可能与预估存在偏差，导致费用计算不准确。
- 亚马逊的FBA费用标准会定期调整（通常每年1月发布调整方案），该技能使用的费用标准基于最近一次公布的费率表，实际执行时可能存在差异。
- 入库运输费用的估算基于合作承运人的标准费率，实际费用可能受到燃油附加费、偏远地区附加费和淡旺季费率浮动的影响。
- 长期仓储费的预估基于假设的库存周转速度，如果实际销售速度低于预期，长期仓储费的影响将显著增加。
- 对于特殊品类商品（如危险品、需要特殊处理的商品、温度敏感商品等），可能存在额外的处理费用，这些费用未包含在标准计算模型中。
- 跨国FBA（如使用欧洲FBA PAN-EU）的费用结构更加复杂，涉及各国不同的VAT税率、跨境配送费和进口关税，需要针对具体运营模式进行单独计算。
