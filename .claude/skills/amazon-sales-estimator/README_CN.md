# Amazon 销量估算工具 — 原理详解

## 概述

本工具通过 Amazon 公开的 BSR（Best Seller Rank，畅销排行榜）数据，结合品类系数和站点系数，估算商品的月销量和月销售额。支持三种模式：BSR计算器、ASIN查询、关键词市场分析。

## 核心原理

### 1. BSR 与销量的反比关系

Amazon 的 BSR 是一个**实时更新的排名数字**，反映商品在其品类中的相对销量。核心规律：

```
BSR越低 → 销量越高
BSR越高 → 销量越低

关系曲线呈幂律分布（长尾效应）：
  BSR #1 的销量远超 BSR #100
  BSR #100 的销量远超 BSR #1000
  以此类推，非线性衰减
```

### 2. BSR 转销量的参考表（美国站 Home & Kitchen 基准）

| BSR 区间 | 估算月销量 | 日均销量 |
|----------|-----------|----------|
| 1-100 | 3,000 - 10,000+ | 100-330+ |
| 100-500 | 1,000 - 3,000 | 33-100 |
| 500-2,000 | 300 - 1,000 | 10-33 |
| 2,000-10,000 | 100 - 300 | 3-10 |
| 10,000-50,000 | 30 - 100 | 1-3 |
| 50,000-100,000 | 10 - 30 | <1 |
| 100,000+ | < 10 | 极低 |

**注意**：这些是估算值，非精确数据。Amazon 不公开实际销量，所有第三方工具（包括本工具）都是基于统计模型估算。

### 3. 品类系数调整

不同品类的销量基数差异很大，通过品类系数修正：

| 品类 | 系数 | 原因 |
|------|------|------|
| 电子产品 | 1.2x | 高客单价、购买频次高，整体销量大 |
| 家居厨房 | 1.0x | 基准品类 |
| 玩具 | 1.1x | 季节性波动大，但高峰期销量很高 |
| 运动户外 | 0.9x | 中等销量基数 |
| 服装 | 0.8x | SKU碎片化严重，单个SKU销量分散 |
| 图书 | 0.7x | 长尾极长，头部集中度低 |

**计算公式**：
```
估算月销量 = 基准销量 × 品类系数
```

### 4. 站点系数调整

不同 Amazon 站点的市场规模差异巨大：

| 站点 | 系数 | 相对于美国站的比例 |
|------|------|-------------------|
| US | 1.0x | 100%（全球最大） |
| JP | 0.6x | 60% |
| DE | 0.5x | 50% |
| UK | 0.4x | 40% |
| CA | 0.3x | 30% |
| 其他 | 0.2x | 20% |

**计算公式**：
```
估算月销量 = 基准销量 × 品类系数 × 站点系数
```

### 5. 三种工作模式

**模式 A — BSR 计算器**：直接输入 BSR 值，通过查表+系数计算得出销量估算。

```
输入: BSR=1500, 品类=Home & Kitchen, 站点=US, 价格=$24.99
  → 查表: BSR 500-2000区间 → 300-1000台/月 → 取中值 450台
  → 品类系数: Home & Kitchen = 1.0x → 450台
  → 站点系数: US = 1.0x → 450台
  → 月收入: 450 × $24.99 = $11,246
```

**模式 B — ASIN 查询**：通过抓取 Amazon 商品页面获取 BSR、价格、品类等信息，然后应用模式 A 的公式自动计算。

```
输入: ASIN B09V3KXJPB
  → 抓取页面 → 提取 BSR、价格、品类
  → 自动应用计算公式
  → 输出: 月销量、月收入、排名百分位
```

**模式 C — 关键词市场分析**：搜索关键词下的多个商品，逐一估算销量后汇总，得出整个细分市场的规模和竞争格局。

```
输入: 关键词 "yoga mat"
  → 搜索 → 获取前10-20个商品的BSR
  → 逐一估算每个商品销量
  → 汇总:
    ├── 市场总销量
    ├── Top 3/Top 10 占比
    ├── 长尾占比
    ├── 价格分布
    └── 市场集中度判断（碎片化 vs 垄断化）
```

### 6. 市场集中度分析

通过分析销量在各卖家之间的分布，判断市场机会：

```
Top 3 占比 < 30% → 碎片化市场，新卖家有机会进入
Top 3 占比 30-50% → 中等集中，需差异化竞争
Top 3 占比 > 50% → 高度集中，头部垄断，新卖家难度大
```

### 7. 价格带分析

统计各价格区间的销量占比，找到"甜蜜点"（Sweet Spot）— 销量最大且竞争相对温和的价格区间。

## 数据来源

本技能的核心数据来源是通过 `web_fetch` 抓取Amazon商品页面获取BSR（Best Seller Rank）、价格、品类等公开数据，然后应用内置的BSR-销量对照表和品类/站点系数进行销量估算。BSR-销量对照表以美国站Home & Kitchen品类为基准，将BSR划分为7个区间（1-100、100-500、500-2000、2000-10000、10000-50000、50000-100000、100000+），每个区间对应一个估算月销量范围。品类系数表包含7个品类的修正系数（电子产品1.2x、家居厨房1.0x、玩具1.1x、运动户外0.9x、服装0.8x、图书0.7x），站点系数表包含7个站点的市场规模比例（US 1.0x、JP 0.6x、DE 0.5x、UK 0.4x、CA 0.3x等）。

市场分析模式（模式C）使用 `web_search` 搜索Amazon上的商品（`site:amazon.com`），逐一获取搜索结果中各商品的BSR数据，再应用上述公式汇总计算市场总销量、Top卖家占比和市场集中度。用户在使用BSR计算器模式（模式A）时需直接输入BSR值、品类和站点信息。整体内置约20条参考数据，所有数据均为固定快照，不会自动更新。

## 局限性

- BSR 转销量是**估算值**，Amazon 不公开实际销量数据
- BSR 会随时间波动（每小时更新），单次快照可能不代表平均水平
- 无法获取历史 BSR 趋势（需要第三方追踪工具）
- 品类系数和站点系数是粗略的统计平均值
- 不考虑季节性波动（Q4销量通常会大幅增长）
- ASIN 查询依赖 Amazon 页面结构，页面改版可能导致抓取失败
