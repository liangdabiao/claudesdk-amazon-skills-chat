# 亚马逊产品摄影规划 (Amazon Product Photography)

## 概述

亚马逊产品摄影规划技能帮助卖家制定完整的产品图片拍摄方案，从拍摄清单的创建到信息图的设计规划，从生活场景图的布景指导到图片优化的技术规范，全方位提升产品图片的视觉吸引力和转化效能。在亚马逊平台，产品图片是买家了解产品的第一媒介，直接影响点击率和转化率。数据显示，高质量的产品图片可以将转化率提升 20%-40%，而图片质量不佳则是买家放弃浏览的首要原因之一。

本技能的核心理念是"每一张图片都要有明确的使命"。亚马逊 Listing 通常支持 7-9 张图片，每张图片都应该承担特定的沟通任务：主图负责在搜索结果中吸引点击，第二张图展示核心卖点，第三张图提供尺寸参考，以此类推。本技能帮助卖家为每张图片定义清晰的视觉目标，确保整个图片序列构成一个完整且有说服力的产品叙事。

## 核心原理

**视觉层级与注意力引导：** 买家在浏览亚马逊搜索结果时，每个产品 Listing 的平均停留时间仅为 2-3 秒。在这极短的时间内，图片必须能够迅速传达产品的核心价值主张。本技能基于视觉层级原理，指导卖家在图片中建立清晰的视觉优先级：最重要的产品特征占据画面最大面积和最突出位置，次要信息通过辅助图形和文字标注补充，整体构图引导买家的视线按照"产品主体 -> 核心卖点 -> 使用场景 -> 信任元素"的路径移动。

**亚马逊图片规范的合规性框架：** 亚马逊对产品图片有严格的技术规范要求，违反规范可能导致图片被拒审或 Listing 被抑制。核心规范包括：主图必须为纯白背景（RGB 255,255,255）、产品占据画面至少 85% 的面积、图片至少 1000x1000 像素（推荐 2000x2000 以支持缩放功能）、不允许添加文字水印和促销标识、辅助视图最多支持 8 张等。本技能将合规性检查融入图片规划的每一个环节，确保生成的拍摄方案从源头上符合平台要求，避免后期返工。

**信息图设计的认知负荷管理：** 信息图（Infographic）是亚马逊 Listing 中展示产品卖点和规格的重要图片类型。本技能基于认知负荷理论，指导信息图的设计遵循"少即是多"的原则：每张信息图聚焦一个核心卖点而非堆砌所有特征，使用图标和简洁文字替代大段描述，采用一致的视觉风格和配色方案建立品牌识别。一张好的信息图应该让买家在 2 秒内理解一个关键卖点，而非让买家花 10 秒去解读一张信息过载的复杂图片。

**生活场景图的情感共鸣机制：** 生活场景图通过展示产品在真实使用环境中的状态，帮助买家建立对产品的情感连接和使用场景想象。与白底图传递的"这是什么"不同，生活场景图传递的是"用起来什么样"。本技能基于场景营销原理，指导卖家规划生活场景图的拍摄要素，包括场景选择（与产品使用环境高度相关的真实场景）、模特选角（与目标受众画像匹配的模特）、道具搭配（强化产品使用效果的辅助道具），以及光线和色调的把控（营造与产品调性一致的氛围感）。

**移动端优先的图片优化策略：** 亚马逊平台超过 70% 的购物流量来自移动端。移动设备的屏幕尺寸限制意味着图片在小屏幕上的表现至关重要。本技能基于移动端视觉优化原则，建议卖家在图片规划中考虑以下要素：关键信息和产品主体在缩小到手机屏幕尺寸后仍然清晰可辨；文字标注的字号足够大以确保移动端可读性；竖版构图比横版构图更适合移动端的浏览习惯；图片序列的整体叙事在移动端快速滑动浏览时依然连贯流畅。

## 工作流程

**第一步：需求分析与产品理解。** 技能从卖家提供的信息中提取产品类型、类目特点、目标受众画像、核心卖点列表、现有图片状况（如是否有旧图需要升级）、拍摄预算范围，以及是否需要同时适配其他平台（如 Shopify 独立站）。

**第二步：补充信息收集。** 通过一次性多选题，技能进一步了解产品尺寸与材质特性（影响拍摄难度和布景方案）、品牌视觉规范（Logo、品牌色、字体等）、竞争对手的图片策略参考、模特需求、特殊拍摄需求（如微距拍摄、动态拍摄、季节性场景等），以及卖家倾向的拍摄方式（自主拍摄或外包给专业摄影团队）。

**第三步：完整拍摄方案生成。** 技能基于收集的信息生成详细的产品摄影拍摄方案，包含：每张图片的拍摄目标说明（如"主图：纯白背景，展示产品全貌，突出品牌 Logo 位置"）、具体的技术参数建议（构图角度、光线方案、背景选择）、信息图的内容规划（每张信息图展示的卖点、文字内容建议、图标风格）、生活场景图的场景描述（场景环境、模特动作、道具清单）、以及尺寸对比图和包装展示图的拍摄指导。

**第四步：预算规划与执行建议。** 根据拍摄方案的复杂度，技能提供预算估算和执行路径建议：对于自主拍摄的卖家，列出需要的设备清单和拍摄技巧提示；对于外包拍摄的卖家，提供摄影团队筛选标准、报价参考范围和交付验收要点；同时提供图片后期处理的优化建议，包括压缩格式选择、Alt 文字编写指南等。

## 数据来源

无外部数据源。本技能的拍摄规划基于AI内置知识，包括Amazon商品图片合规要求的公开文档（如主图白底规范、像素要求、禁止添加的元素等）以及产品摄影领域的专业最佳实践。不依赖任何外部API或第三方数据服务。用户在使用本技能时需要提供产品类型、尺寸材质特性、拍摄预算和预期用途等信息，以便生成切实可行的摄影拍摄方案。

## 局限性

本技能提供的是摄影规划和设计指导方案，而非实际执行拍摄服务。最终的图片质量高度依赖摄影师的技术水平、设备条件和后期制作能力。不同产品类目对摄影的要求差异巨大，如食品类目需要专业的食物造型师，服装类目需要经验丰富的时尚摄影师，电子类目可能需要特殊的灯光和微距设备，通用建议的适用性需要根据具体产品进行调整。亚马逊的图片规范可能在不同类目和不同站点之间存在细微差异，卖家需要在实际上传前参照最新的官方规范进行最终确认。生活场景图的拍摄涉及模特、场地和道具等多方面的协调，执行复杂度和成本可能超出预期。自主拍摄的卖家需要一定的摄影基础和学习成本，而专业摄影的外包费用对于预算有限的中小卖家可能是一笔不小的开支。此外，图片只是 Listing 优化的一个维度，即使图片质量优秀，如果产品标题、五点描述和价格等要素不到位，整体转化效果仍然可能受限。
