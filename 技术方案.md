# AmazonSkillsChat — AI 亚马逊卖家全链路助手 技术方案

## 1. 项目概述

基于 **Claude Agent SDK** 构建一个对话式 Web 应用，整合 **Amazon-Skills-n/** 的 54 个亚马逊卖家技能，提供从选品调研、关键词研究、Listing 优化、FBA 费用计算到 PPC 广告投放的全链路 AI 辅助。

**项目目录**: `D:/claudesdk-test/amazon-skills-chat/`
**端口**: 3014

## 2. 架构设计

```
用户浏览器 ←WebSocket→ Express Server ←→ Claude Agent SDK ←→ Amazon Skills (54个)
                          │                              │
                    reports/                    Playwright MCP + WebSearch
                   (分析报告)                  + Bash/Python 脚本
```

### 核心组件

| 组件 | 技术 | 说明 |
|------|------|------|
| 后端 | Express + WebSocket + claude-agent-sdk | 复用 review-chat / geo-chat 成熟架构 |
| 前端 | React 18 + Tailwind CSS 4 + Vite 6 | Amazon 橙色主题 |
| AI 引擎 | Claude Agent SDK v0.1.11 | `query()` + MessageQueue 模式 |
| 技能系统 | `.claude/skills/` 目录 | 54 个 Amazon Skills（从 Amazon-Skills-n/ 复制） |
| 浏览器自动化 | Playwright MCP | 产品页面抓取、竞品分析 |
| 脚本执行 | Bash + Python 3 | 关键词研究、FBA/关税计算 |

## 3. Skills 技术方案（核心）

### 3.1 技能清单

**生产就绪（8 个）：**

| 技能 | 功能 | 所需工具 | 输入 → 输出 |
|------|------|----------|------------|
| `amazon-one-shot` | 一站式深度调研 | Playwright + Bash | ASIN → 9 份报告 |
| `amazon-category-research` | 品类深度调研 | Playwright + Agent | Best Sellers URL → 5 份报告 |
| `amazon-store-research` | 店铺深度调研 | Playwright + Agent | Store URL → 5 份报告 |
| `amazon-keyword-research` | 关键词挖掘 | Bash 脚本 (curl) | 关键词 → 长尾词 + 机会评分 |
| `amazon-listing-optimization` | Listing 优化 | Read + Write | 关键词/竞品ASIN → 优化文案 |
| `amazon-fba-calculator` | FBA 费用计算 | Python 脚本 | 产品参数 → 费用明细 |
| `amazon-ppc-campaign` | PPC 广告结构 | Read + Write | 产品+预算 → 广告结构 |
| `amazon-sales-estimator` | 销量估算 | Read + Write | BSR/ASIN → 月销量 |
| `tariff-calculator-amazon` | 关税计算 | Python 脚本 | 贸易路线 → 关税+到岸成本 |

**Beta 技能（25 个，一并集成）：**

- 产品研究：trending-products, product-research, niche-finder, seller-analytics
- Listing：a-plus-content, backend-keywords, search-optimization, listing-images
- 竞品：competitor-analysis, brand-analytics, review-analyzer
- 定价：profit-analyzer, repricing-strategy, buy-box, deal-finder, shipping-calculator
- 广告：advertising-strategy, negative-keywords, display-ads
- 分析：rank-tracker, keyword-tracker, price-tracker, product-photography
- 扩展：global-selling, fba-prep

## 9. 与现有 amazon-chat 的区别

| 维度 | 现有 amazon-chat (端口 3005) | 新项目 amazon-skills-chat (端口 3014) |
|------|------|------|
| 技能来源 | 5 个自建技能 | **54 个 Amazon-Skills-n 社区技能** |
| 数据源 | Sorftime MCP API (34 接口) | **Playwright 浏览器自动化 + Bash/Python** |
| 浏览器 | 无 | **Playwright MCP（产品抓取）** |
| 脚本 | Python 35+ 脚本 | **Bash (keyword) + Python (FBA/tariff)** |
| 功能 | 竞品/品类/关键词/选品/评论 | **全链路 54 技能覆盖** |

## 10. 关键注意事项

1. **Playwright 必需** — amazon-one-shot / category / store 三个核心技能依赖浏览器自动化
2. **Python 3 环境** — FBA 计算器和关税计算器使用 Python 3 stdlib
3. **Bash 脚本** — 关键词研究使用 curl 调用 Amazon autocomplete
4. **报告目录扫描** — 技能输出到 `reports/` 和 `.claude/skills/*/output/`，服务器需同时扫描
5. **model 显式设置** — `model: process.env.MODEL || "sonnet"` 必须指定
6. **distPath 修复** — 生产构建需处理 `__dirname` 路径兼容（使用 existsSync 判断）
